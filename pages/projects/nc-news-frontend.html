NC News Front-end

This project is the front-end of the NC News application, built as part of my Northcoders coding bootcamp. The app is a responsive news web application that allows users to browse, read, and interact with articles on a variety of topics. It connects to the NC News API to provide dynamic content and real time user interactions.

The focus of the project was creating a clear and intuitive interface for content heavy information, ensuring users can easily filter articles, read comments, and interact through voting and discussion.

Features

  • Browse articles with filtering by date or votes
  • Read full articles with clearly structured content and comments
  • Post, delete, and vote on comments with immediate visual feedback
  • Upvote and downvote functionality for articles and comments
  • Seamless, mobile-friendly responsive design
  • Integration with the NC News backend API
  • Optimised for performance and smooth UX

Challenges & Learnings

Building the front-end helped me understand how interface decisions affect usability, particularly when working with dynamic data. I focused on handling loading, error, and empty states clearly so users always understood what was happening. Designing the layout for different screen sizes reinforced the importance of hierarchy, spacing, and readability in content heavy interfaces. Integrating user interactions such as voting and commenting highlighted the value of immediate feedback in creating a responsive and engaging user experience.
