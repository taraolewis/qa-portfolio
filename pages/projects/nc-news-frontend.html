NC News Front-end

This project is the frontend of the NC News application, built during my Northcoders bootcamp. It connects to a RESTful API to fetch and display articles, comments, and user data. The application allows users to browse content, sort articles, and interact through voting and commenting. A key focus was handling different application states clearly, including loading, errors, and empty results. I worked on ensuring the interface updated reliably when data changed and that user actions triggered predictable updates in the UI.

Features

  • Fetched and displayed dynamic data from a RESTful API
  • Implemented voting, posting, and deleting comments with real time UI updates
  • Added filtering and sorting functionality for articles
  • Handled loading, error, and empty states to improve reliability
  • Built a responsive layout for desktop and mobile

What I Learned

This project helped me understand how frontend applications interact with backend systems and manage changing data. I improved my confidence working with asynchronous requests, managing state in React, and building interfaces that remain stable even when requests fail or data is delayed.
