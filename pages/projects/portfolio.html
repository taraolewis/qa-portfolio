Portfolio Website

A responsive, multi-page portfolio designed to clearly present my projects, design work, and technical background. The site was planned using Figma wireframes to explore layout, content structure, and visual hierarchy before being designed and developed with a focus on clarity, accessibility, and usability.

Features

  • Clear content structure to help users quickly understand my work and skills
  • Responsive layout optimised for desktop and mobile
  • Consistent visual hierarchy, spacing, and typography
  • Accessible front-end practices using semantic HTML
  • Simple, intuitive navigation across pages
  • Deployed live for real life use and iteration

Challenges & Learnings

Building my portfolio strengthened my ability to translate wireframes into a finished interface while maintaining consistency across pages. Iterating on layout and spacing improved my understanding of responsive design, accessibility considerations, and how presentation impacts usability.

Tech Used
