Portfolio Website

A responsive multi page website built to showcase my projects and technical skills. The site was planned using Figma wireframes and then developed using semantic HTML, CSS, and JavaScript.

The focus was on building a clear structure, reliable navigation, and consistent behaviour across devices and screen sizes. The site is deployed live and updated through Git based version control.

Features

  • Multi page structure with simple, clear navigation
  • Responsive layout for desktop and mobile
  • Semantic HTML for accessibility and maintainability
  • Reusable layout patterns for consistency
  • Deployed via Netlify

What I Learned

Building and deploying this site improved my understanding of how frontend applications behave in real environments. It strengthened my approach to layout structure, browser compatibility, and maintaining clean, readable code.

Tech Used

HTML CSS JavaScript Figma Netlify
