Co-Write: Collaborative Story Writing App

Co-Write is a mobile application developed as a group project at Northcoders Bootcamp. It allows users to collaboratively create stories by starting new threads, contributing to existing ones, and viewing completed stories with AI-generated illustrations. This MVP represents our first major React Native application after 3 months of JavaScript learning and 3 weeks of focused development.

Features

  • User authentication with Firebase
  • Create new story threads and contribute collaboratively
  • AI-powered story continuation using OpenAI & OpenRouter APIs
  • View completed stories with AI-generated images
  • Real time updates for collaborative editing
  • Word limit enforcement and voting system
  • Mobile first responsive design with React Native

Tech Used

React Native TypeScript Firebase React Navigation OpenAI API OpenRouter API
