React Native Calculator App

A simple, responsive calculator built with React Native and Expo. Designed with clean UI, intuitive interactions, and a focus on mobile usability.

Features

  • Basic arithmetic operations (add, subtract, multiply, divide)
  • Clear layout and readable visual hierarchy
  • Responsive layout across devices
  • Touchable, mobile-friendly buttons
  • Built with React Native + Expo
  • Immediate feedback when entering values and actions
  • Clear and undo actions to reduce mistakes

Challenges & Learnings

Building this project helped me think about UX in repeated, everyday interactions, especially tap targets, spacing, and visual feedback. Iterating on layout improved my understanding of mobile interface hierarchy, while implementing input and calculation logic reinforced the importance of predictable behaviour and preventing user error.

Tech Used
