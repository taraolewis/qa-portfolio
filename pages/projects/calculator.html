React Native Calculator App

A mobile calculator built with React Native and Expo. This project focuses on logical accuracy, input handling, and predictable behaviour across a range of user interactions. It was a useful space to practice validating requirements, checking edge cases, and ensuring consistent results.

Testing and Behaviour Focus

  • Validated calculation logic for common and edge case inputs
  • Handled repeated taps, rapid input changes, and unexpected sequences
  • Ensured clear and undo actions reset state consistently
  • Verified behaviour for divide by zero and invalid operations
  • Checked display formatting for long numbers and decimal input
  • Manually tested across device sizes to confirm layout stability

Features

  • Basic arithmetic operations including add, subtract, multiply, and divide
  • Clear layout with readable hierarchy
  • Responsive layout across devices
  • Touchable buttons designed for mobile use
  • Immediate feedback during input and actions
  • Clear and undo actions to reduce mistakes

Challenges and Learnings

This project reinforced the importance of testing logic with a wide range of inputs and interaction sequences, not only expected usage. It improved my ability to think in terms of boundary cases, state management, and predictable outcomes, which are central to delivering reliable software.
