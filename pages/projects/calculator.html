React Native Calculator App

A mobile calculator built with React Native and Expo. This project focused on building the core calculation logic and managing state so results update correctly as users interact with the app. The goal was to ensure the app behaved consistently and predictably.

Features

  • Basic arithmetic operations (add, subtract, multiply, divide)
  • Managed user input and state updates
  • Reusable button components for mobile interaction
  • Responsive layout for different screen sizes

What I Learned

This project improved my understanding of state management and logical flow in a React Native application. It helped me think more carefully about how user input affects application behaviour.

Tech Used

React Native Typescript Expo JavaScript Expo Router
