NC News API Back-end

This project is the RESTful API for the NC News application, built during my Northcoders bootcamp. I created endpoints to manage articles, topics, comments, and users, using PostgreSQL to store and retrieve data. The API supports filtering and sorting articles, adding and deleting comments, and returning structured responses with clear error handling. I used a test-driven approach with Jest and Supertest to make sure each endpoint behaved as expected and handled invalid requests properly.

Features

  • Built RESTful endpoints for articles, comments, topics, and users
  • Wrote SQL queries to manage relational data
  • Implemented filtering, sorting, and input validation
  • Deployed a live version using Render

What I Learned

This project helped me understand how backend systems are structured and how APIs connect to a frontend application. It strengthened my confidence working with databases, handling errors, and building reliable endpoints.

Tech Used

Node.js Express PostgreSQL Jest Supertest REST API Render
