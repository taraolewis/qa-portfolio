Mini Design System

A small design system created in Figma to explore reusable UI components, interaction states, and layout patterns for a data-driven dashboard. The project focuses on consistency, clarity, and scalability rather than visual branding.

Features

  • Core UI components including buttons, inputs, dropdowns, and modals
  • Defined component states (default, hover, focus, error, disabled)
  • Shared typography, colour, and spacing foundations
  • Clear component naming and organisation within Figma
  • Basic usage guidelines and documentation

Challenges & Learnings

Creating the design system helped me think beyond individual screens and focus on how components work together as part of a wider system. I learned how small design decisions around spacing, states, and naming impact usability, consistency, and collaboration with developers.
